Download Panacea

Requires Windows 10 and above

Scroll down for installation steps

Installation

  • Download PanaceaSetup.exe
  • Place the setup file in the directory you wish to install
  • The installer will prompt you to download Ollama if it doesn't find OllamaSetup.exe
    (you may skip this step if you already have Ollama installed)
    • Note: If you haven't installed Ollama before, you will also have to install an AI Model. Refer to ollama.com for steps to install it
  • Once finished, you will see a bin folder in the same directory as the setup file

Runtime

  • Start the AI Model either by using the model_config.exe in the bin folder or by typing "ollama run [your model name] in a terminal window
    • If using the model_config.exe in the bin folder, kindly keep it open in the background for convenience
  • Use the model_config.exe file to select the AI model for Panacea and check if it's running
  • Run the .exe file of the flavour of Panacea you wish
    • Currently, on-the-fly AI model switching is not supported, but you can check the model being used in the app
  • Once done, you can stop the AI model using model_config.exe or by typing "ollama stop [your model name]" in the terminal window
